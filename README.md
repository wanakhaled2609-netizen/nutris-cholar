## 💻 الحسابات المستخدمة

### 1. مؤشر كتلة الجسم (BMI)
```
BMI = الوزن (كجم) / (الطول (م))²
- < 18.5: نحافة
- 18.5-25: وزن مثالي
- 25-30: زيادة وزن
- > 30: سمنة
```

### 2. الوزن المثالي (IBW)
```
للذكور: الطول - 100 - (الطول - 150) / 4
للإناث: الطول - 100 - (الطول - 150) / 2
```

### 3. معدل الأيض الأساسي (BMR) - Harris-Benedict
```
للذكور: (10 × الوزن) + (6.25 × الطول) - (5 × العمر) + 5
للإناث: (10 × الوزن) + (6.25 × الطول) - (5 × العمر) - 161
```

### 4. السعرات اليومية (TDEE)
```
TDEE = BMR × مستوى النشاط
```

- الدهون: 30%
- **الألياف: > 30 غرام/اليوم**

### أمراض القلب

### api.php
يوفر الدوال التالية:

#### calculateNutrition()
```php
POST /api.php?action=calculateNutrition
Parameters: weight, height, age, gender, disease, activity
Returns: BMI, BMR, TDEE, macronutrients
```

#### generateMeals()
```php
POST /api.php?action=generateMeals
Parameters: disease, tdee, gender
Returns: breakfast, lunch, dinner, snacks
```

#### savePlan()
```php
POST /api.php?action=savePlan
Parameters: name, age, weight, height, disease, etc.
Returns: success, planId
```

#### validateInputs()
```php
POST /api.php?action=validateInputs
Parameters: weight, height, age, gender, activity
Returns: validation result with errors
```
