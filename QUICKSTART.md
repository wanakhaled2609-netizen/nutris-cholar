# 🚀 دليل البدء السريع - NutriScholar

## ⚡ الخطوات الأساسية (3 خطوات فقط)

### 1️⃣ نسخ الملفات
```bash
# انسخ جميع الملفات إلى مجلد في خادم الويب
cp -r ./* /var/www/html/nutrischolar/
```

### 2️⃣ تعديل الأذونات (Linux/Mac)
```bash
chmod 755 api.php
mkdir -p saved_plans
chmod 777 saved_plans
```

### 3️⃣ فتح التطبيق
```
http://localhost/nutrischolar/index.html
```

## 🐛 استكشاف الأخطاء

### المشكلة: الصفحة لا تحمل
**الحل:**
```
1. تحقق من أن الملف موجود في المسار الصحيح
2. تأكد أن خادم الويب يعمل
3. جرب متصفح آخر
```

### المشكلة: الحسابات لا تعمل
**الحل:**
```
1. تأكد أن php مثبت: php -v
2. تحقق من الـ console للأخطاء: F12
3. جرب الحسابات المحلية (بدون API)
```

### المشكلة: PDF لا يتم حفظه
**الحل:**
```
1. تحقق من الإنترنت
2. السماح للمتصفح بـ JavaScript
3. تجربة متصفح آخر (Chrome أفضل)
```

## 📊 أمثلة على البيانات

### مثال 1: شخص سليم
```
الاسم: محمد علي
العمر: 30 سنة
الوزن: 75 كجم
الطول: 175 سم
الجنس: ذكر
الحالة: شخص صحي
النشاط: متوسط

النتيجة:
- BMI: 24.5 (وزن مثالي)
- السعرات: 2400 سعرة/اليوم
```

### مثال 2: مريض سكري
```
الاسم: فاطمة أحمد
العمر: 45 سنة
الوزن: 85 كجم

الجنس: أنثى
الحالة: مريض سكر (النوع الثاني)
النشاط: خفيف

النتيجة:
- BMI: 31.2 (سمنة)
- السعرات: 1950 سعرة/اليوم

```

## 🎓 موارد مفيدة

### الحسابات المستخدمة:
- 📊 مؤشر كتلة الجسم (BMI)
- 🔥 معدل الأيض الأساسي (BMR)
- 💪 السعرات اليومية (TDEE)

### المعادلات:
```
BMI = الوزن / (الطول)²
BMR = Harris-Benedict Equation
TDEE = BMR × مستوى النشاط
```
